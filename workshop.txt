description=
description=This mod implements a robust vehicle claiming system with proper client-server architecture, preventing cheating through server-side validation and providing a clean UI for vehicle management.
description=
description=Server-Authoritative Design
description=All critical operations (claiming, releasing, access control) are validated and executed exclusively on the server. Clients send requests, but the server has final authority over all state changes.
description=
description=Anti-Cheat Measures
description=Steam ID Verification: Server validates that the requesting player's Steam ID matches the command
description=Proximity Checks: Server verifies player is within range before allowing actions
description=Ownership Validation: All modifications require proof of ownership or admin privileges
description=ModData Sync: Vehicle claim state is stored in vehicle ModData and synchronized by the game engine
description=
